FAQ

Common questions

Is this connected to a real credit bureau?
No. This is an educational/portfolio project trained on a synthetic dataset that mirrors real-world credit approval schemas. It is not connected to any credit bureau and should not be used for real lending decisions.
Which model is currently deployed?
The model with the highest ROC-AUC on the held-out test set is selected automatically by train.py and saved to model/model.pkl. Check the Dashboard page for the current leaderboard.
Can I retrain it on my own data?
Yes — drop a CSV matching the documented schema into the dataset/ folder and run python train.py. Every artifact (model, scaler, encoders, metrics) is regenerated automatically.
Is there an API, or only the web form?
Both. POST /api/predict accepts JSON and returns the same decision payload the web form uses — see the README for the full request/response schema.
